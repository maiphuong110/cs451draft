# My Website Project

Đây là dự án demo để thực hành quy trình làm việc với Git, GitHub và Nginx.

## Các file hiện có:
* `index.html`: Giao diện trang chủ cơ bản.
* `index.php`: Trang kiểm tra thời gian thực để test cấu hình PHP trên server.
* `README.md`: File tài liệu mô tả dự án.

## Công cụ sử dụng:
- Git & GitHub
- Vim Editor
- Nginx Web Server
